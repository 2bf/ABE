ABE - Languages

ABE now supports over 100 languages, making it a much more accessible tool for all users.

ABE uses AI to automatically detect language.
All you have to do is talk to ABE with any of the supported languages below, and it will respond with the language used!

Note: ABE works best with English. Some languages haven't been tested yet.
